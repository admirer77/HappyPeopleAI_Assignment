Table Layout: fixed vs auto

table-layout: auto; (Default Behavior)

table-layout: fixed;

Table with table-layout: fixed;
Short Header A Header with Slightly Longer Text Another Header

Data 4 This cell has a very long piece of content that will be truncated in 'fixed' layout. Data 6
Data 7 Data 8 Data 9 with more text

In the auto layout, the second column expands to fit the long content. In the fixed layout, the column widths are determined by the table width and the first row, and the long content is truncated (due to overflow: hidden; and text-overflow: ellipsis;).
