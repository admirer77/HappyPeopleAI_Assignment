This is the main content area below the hero image. A hero image is a large banner image prominently placed on a web page, typically at the top. It's designed to grab attention and convey the main message or theme of the page.

Using background-size: cover; ensures the image scales to cover the entire container while maintaining its aspect ratio. Setting a height on the container (like 60vh) controls how much vertical space the hero image occupies.
