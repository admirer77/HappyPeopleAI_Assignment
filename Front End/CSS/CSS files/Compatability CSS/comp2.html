list-style-type Examples

Unordered Lists (<ul>)

list-style-type: disc; (Default)

  • Item 1
  • Item 2
  • Item 3

list-style-type: circle;

  • Item A
  • Item B
  • Item C

list-style-type: square;

  • Item X
  • Item Y
  • Item Z

list-style-type: none;

  • Item without marker
  • Another item

list-style-image: url(...);

  • Item with custom image marker
  • Another item

Ordered Lists (<ol>)

list-style-type: decimal; (Default)

  1. First Step
  2. Second Step
  3. Third Step

list-style-type: decimal-leading-zero;

  1. Task One
  2. Task Two
  3. Task Three

list-style-type: lower-alpha;

  1. Point a
  2. Point b
  3. Point c

list-style-type: upper-roman;

  1. Chapter I
  2. Chapter II
  3. Chapter III
